# Interview Scheduler

Interview Scheduler is a simple, single-page student-instructor appointment scheduler made by Graham Mothersill, a web-dev student at Lighthouse Studios.

Built using React.

## Setup

Install dependencies with `npm install`.

```sh

```

```sh

```
